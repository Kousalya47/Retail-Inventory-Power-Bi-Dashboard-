# Retail-Inventory-Power-Bi-Dashboard-
 Retail Inventory and Sales Analysis Dashboard using Power BI

## Tool Used
- Microsoft Power BI

## Project Description
This project presents an interactive Power BI dashboard for analyzing retail
inventory and sales performance. The dashboard provides key insights into
sales trends, stock levels, and reorder requirements.

## Data Used
- Product details (category and pricing)
- Store information (location-wise performance)
- Inventory data (opening stock, closing stock, reorder level)
- Sales data (date-wise quantity and revenue)

## Dashboard Features
- KPI cards for Total Sales, Quantity Sold, Closing Stock, and Low Stock Products
- Category-wise and store-wise sales analysis
- Inventory monitoring and reorder alerts
- Interactive slicers for category, product, store, and city
- Single-page dashboard design
